# Mi_tema

Tema personalizado para Shopify. 

## Descripción
Este repositorio contiene un tema desarrollado para Shopify, listo para ser editado en VS Code, versionado en GitHub y sincronizado con una tienda Shopify usando Shopify CLI.

## Instrucciones rápidas

1. Clona este repositorio o trabaja directamente desde VS Code.
2. Usa Shopify CLI para conectar y sincronizar con tu tienda.
3. Realiza cambios, haz commit y push a GitHub.
